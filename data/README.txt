Drop your Excel (.xlsx) comparison files here.

Format:
  Column A = Site A URL  (e.g. https://staging.example.com/page)
  Column B = Site B URL  (e.g. https://www.example.com/page)

- A header row (e.g. 'Site A' / 'Site B') is fine, it will be skipped.
- Both URLs must start with http:// or https://

Then double-click run.bat to start.
